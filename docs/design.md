# otakekei.com リニューアル設計 (2026-06-10)

## 目的
2025年選挙時のWixサイトを、現職議員の公式サイトとして全面リニューアル。
参考: triorb.co.jp（黒背景・巨大タイポグラフィ・スクロール演出）

## デザイン方針（案B 承認済み）
- ディープネイビー × サンライズオレンジ（伊東の海とオレンジビーチの朝日）
- 巨大タイポグラフィ、スクロールで現れるセクション、数字のカウントアップ
- 信頼感を保ちつつTriOrb級のモダンさ

## 構成
1. ヒーロー（ポートレート＋巨大コピー「伊東の未来を、切り拓く。」）
2. ごあいさつ（現職向けに文章調整）
3. 政策3本柱（子どもファースト / 公共資産の最適化 / 伊東の魅力発信）
4. Clean Vibes（実績数字 400+回・800m ＋写真グリッド）
5. プロフィール（タイムライン）
6. SNS・お問い合わせ（Instagram / YouTube / Facebook / LINE）

## 技術
- 静的サイト（HTML/CSS/JS、ビルド不要）
- GitHub Pages で公開 → 確認後 otakekei.com のDNSを向け替え
- 写真は旧Wixサイトから流用（assets/img/）
- 公選法配慮: 投票依頼文言なし、活動報告サイトとして構成
